Smart Waste Classifier is an AI-powered waste management system that automatically classifies waste images into three categories: Recyclable, Organic, and Hazardous. The project uses a Convolutional Neural Network (CNN) built with TensorFlow and Keras to analyze uploaded images and predict the appropriate waste category.

The system includes a simple and user-friendly web interface that allows users to upload waste images and instantly receive classification results. By automating waste segregation, the project aims to support sustainable waste management practices, reduce human error, and encourage proper disposal and recycling of waste materials.

The project was trained using publicly available datasets:

- TrashNet Dataset
- Organic Waste Dataset
- E-Waste Dataset
